g0vTxT
=============
Visualize g0v hackpad's information and view g0v communiques.

## Structure

- asserts
    + data
    + imgs
    + scripts
    + styles
- vendor
    + angular
    + d3js
    + jquery
    + semantic-ui
- views

## Installation
[![devDependency Status](https://david-dm.org/g0v/g0vTxT/dev-status.svg?style=flat-square)](https://david-dm.org/g0v/g0vTxT#info=devDependencies)

`npm i`

`npm start`

and then open `http://localhost:3000/`
